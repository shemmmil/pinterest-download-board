# Privacy Policy for Pinterest Board Downloader

**Last Updated:** November 26, 2025  
**Effective Date:** November 26, 2025

## Overview

Pinterest Board Downloader ("the Extension") is committed to protecting your privacy. This Privacy Policy explains how the Extension handles information when you use it.

## Information Collection

### What We DON'T Collect

The Extension **does NOT collect, store, or transmit** any personal information, including but not limited to:

- ❌ Personal identification information
- ❌ Browsing history
- ❌ Email addresses
- ❌ User credentials
- ❌ Analytics or tracking data
- ❌ Cookies or similar technologies

### What We DO Process (Locally Only)

The Extension processes the following data **locally in your browser only**:

- ✅ Current Pinterest board URL (to identify which board to download)
- ✅ Public pin data from Pinterest (images, videos, titles)
- ✅ Downloaded files (stored directly to your Downloads folder)

**Important:** All processing happens entirely within your browser. No data is sent to external servers.

## Data Storage

- **Local Storage:** The Extension does not store any persistent data
- **Downloads:** Files are saved directly to your browser's Downloads folder
- **No Cloud Storage:** No data is uploaded to any cloud service
- **No Database:** No user data is stored in any database

## Data Transmission

- **No External Servers:** The Extension does not communicate with any external servers except Pinterest.com
- **Pinterest API:** The Extension only communicates with Pinterest.com to fetch public board data
- **No Third Parties:** No data is shared with third-party services

## Permissions Explanation

The Extension requires the following permissions:

### `activeTab`
**Purpose:** Access the current Pinterest tab to read board information  
**Data Access:** Only the current Pinterest board URL and page content  
**Usage:** Required to identify which board to download

### `tabs`
**Purpose:** Manage browser tabs  
**Data Access:** Tab URLs (only for Pinterest.com)  
**Usage:** Required for extension functionality

### `sidePanel`
**Purpose:** Display the extension's side panel  
**Data Access:** None  
**Usage:** UI functionality

### `contentSettings`
**Purpose:** Manage content settings  
**Data Access:** None  
**Usage:** Extension configuration

### `host_permissions` (pinterest.com)
**Purpose:** Access Pinterest.com content  
**Data Access:** Public board data (images, videos, titles)  
**Usage:** Required to download pins from Pinterest boards

## Children's Privacy

The Extension does not knowingly collect information from children under 13 years of age. If you are under 13, please do not use this Extension.

## Third-Party Services

The Extension only interacts with:

- **Pinterest.com** - to fetch public board data
- **Your Browser** - to save downloaded files

No other third-party services are used.

## Data Security

- All data processing happens locally in your browser
- No data is transmitted over the network except to Pinterest.com
- Downloaded files are saved directly to your device
- No user data is stored on external servers

## Your Rights
